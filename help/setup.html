Options Page:
Right after your first start you should open the Option page and configure the rules you want to use. There are also options for printing.

Use Feat Rules for Pilots:
This enables/disables the language and feat drop-down-menus in the pilot window.

Use Max. Aircraft Characteristics:
This enables/disables the validation of the maximum aircraft characteristics, which is loaded from the aircraftConfigs.ae file.

Modified Rocket Hardpoint Mass Reduction:
The official rule is that each RHP that is removed, reduces the weight of 10 lbs. The preset custom rules reduces the weight of 125 lbs. (which is the FF5 housrule). You can adjust the custom value if you change the ruleConfigs.ae file. How you do that is described here.

Print Aircraft Flavor Sheet:
If this option is checked, you print also the flavor sheet.

Using the FF5 Loadout Housrule:

If you look into to the "data" sub-directory you will find 3 different baseData files.

  • baseData.ae
  • baseData.orgbackup.ae
  • baseData.ff5backup.ae

    • Only the first one is used currently by Aeronaut. The other two are backups. The two backup files differ only in the rocket section. "orgbackup" uses the official slot spaces for rockets and "ff5back" uses the FF5 houserule.

    So if you want to use the houserule, just delete baseData.ae, copy baseData.ff5backup.ae and rename it to baseData.ae. Now restart Aeronaut and you can add more rockets to your slots.

    But keep in mind:
