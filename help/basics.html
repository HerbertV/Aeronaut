The Work Area

On the left side is the Main Menu, on the bottom is the Toolbar, on the right side are all minimized Windows and in the center is the working area where the active window is located.



The Main Menu

Here you can choose what you want to do. The Pilot Roster opens a window for editing your pilots. Squadrons opens the squadron window. Aircraft Constructor opens the window for editing and designing your aircrafts. Loadouts opens the loadout window.
The Zeppelin Constructor is for designing zeppelins with the FF5 Houserules. It is disabled now and it will be enabled with an upcoming update.



The Toolbar

The Toolbar is linked to the active Window. This means you can only use "New", "Open", "Save" and "Print" if a window is in the main area. The Squadrons and Loadout windows cannot be printed.

New
Clears the active Window.

Open
Loads an aircraft, pilot, etc. in the active window or if no window is open, it opens a new window. While a window is active you can only load the same type the window can display. If no window is active, you can choose which file type you want to open.

Save
Saves the active window. A window can only saved if it is valid. Look for red texts if you see the invalid stamp. This gives you a hint what is wrong.

Save As...
Saves the active window under a new name. A window can only saved if it is valid. Look for red texts if you see the invalid stamp. This gives you a hint what is wrong.

Delete
This function is disabled right now. You can delete a file here.

Print
Prints the active Window. You can only print valid files and only from the aircraft, pilot or zeppelin window.

Options
Opens the option page in the Toolbook. Here you can change some settings.



Shortcuts

  • CTRL + N = Opens new file in active window
  • CTRL + L = Load file in active window
  • CTRL + S = Save active window
  • CTRL + D = Delete file
  • CTRL + P = Print active window
  • CTRL + TAB = Switch active window
  • ESC = Exit Aeronaut
  • F1 = Toggle Console
